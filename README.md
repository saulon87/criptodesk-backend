# CriptoDesk v2

Dashboard web para estrategia Spot administrable en Ethereum.

## Reglas principales

- ETH es el activo principal.
- No utiliza futuros.
- No utiliza margen.
- No utiliza apalancamiento.
- El capital máximo predeterminado es 65 % del saldo disponible.
- El saldo y el porcentaje son editables.
- Si no existe oportunidad clara, recomienda esperar.
- La operación solo se sugiere si la relación riesgo-beneficio es favorable.

## Variables opcionales de OKX

Estas variables solo se usan para consultar saldo. No ejecutan órdenes.

- OKX_API_KEY
- OKX_SECRET_KEY
- OKX_PASSPHRASE
